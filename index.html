Descripción del problema

Los estudiantes de pregrado tienen dificultades para acceder a herramientas integrales y accesibles para prepararse para el examen Saber Pro. La oferta actual suele ser dispersa, poco personalizada o de difícil acceso, limitando una preparación estructurada y realista.

Propuesta: desarrollar una plataforma web que centralice recursos académicos, facilite el estudio autónomo y brinde un entorno flexible y accesible para la preparación del examen Saber Pro, optimizando rendimiento y equidad.

Stack tecnológico

  • Frontend: React.js + TypeScript
  • UI: Bootstrap
  • Backend: Java (Spring Boot sugerido)
  • DB: PostgreSQL
